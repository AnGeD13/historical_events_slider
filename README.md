## Стек
- React
- TypeScript
- Webpack
- SCSS, модульные стили
- gsap - библиотека для анимаций
- Swiper - библиотека для слайдера
- Prettier - для форматирования кода

## Инструкция по запуску проекта

### Установка зависимостей
```bash
npm install
```

### Запуск проекта в режиме разработки
```bash
npm start
```
Приложение будет доступно по адресу http://localhost:3000

### Сборка проекта
```bash
npm build
```

### Проверка кода на соответствие заданному в .prettierrc формату
```bash
npm format:check
```

### Форматирование кода
```bash
npm format
```

## Комментарии
1. Для улучшения доступности приложения к кнопкам добавлены aria-атрибуты и кастомные стили для состояния фокуса.
2. Добавлен хук `useResizeCirclePagination` для автоматического изменения размера блока с круговой пагинацией.
3. В тз есть фраза: "Возможно существование от 2 до 6 временных отрезков". Соответственно, раз количество отрезков строго более одного, то круговая пагинация (на десктопах и планшетах), обычная пагинация (на мобильных устройствах) и блок с переключением отрезков существуют всегда. А размер входных моковых данных пока что задан вручную и равен шести. На данный момент обработки размера входных данных нет. И кажется, что это не имеет смысла, пока кнопки на окружности не станут слишком близкими друг к другу.
4. Также по тз блок должен быть независимым. Именно для этого в App.tsx добавлен фрагмент (`<></>`), внутрь которого можно вставить что-то дополнительно.
